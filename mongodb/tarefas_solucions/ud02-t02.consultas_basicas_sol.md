## 4. Colección `routes`

### a. Todos os documentos (0,25)
Recupera todos os documentos da colección.

```js
db.routes.find()
```
![db.routes.find()](images/ud02-01-04a.png)

### b. Primeiro documento (0,25)
Recupera o primeiro documento da colección.

```js
db.routes.findOne()
```
![db.routes.findOne()](images/ud02-01-04b.png)

### c. Voos con orixe KZN (0,5)
Recupera os documentos que teñen como orixe o aeroporto **"KZN"**  
(campo `src_airport`).
```js
db.routes.find({ src_airport: "KZN" })
```
![src_airport: "KZN"](images/ud02-01-04c.png)


### d. Número de voos desde KZN (1)
Recupera cantos voos saen do aeroporto **"KZN"**.

```js
db.routes.countDocuments({ src_airport: "KZN" })
```
![countDocuments](images/ud02-01-04d.png)

### e. Liña aérea Aerocondor (1)
Recupera os documentos cuxa liña aérea sexa **"Aerocondor"** e **non** saian do aeroporto **"KZN"**.

```js
db.routes.find({
  airline: "Aerocondor",
  src_airport: { $ne: "KZN" }
})
```
![$ne:"KZN"](images/ud02-01-04e.png)

### f. Voos con paradas (1)
Recupera cantos documentos teñen máis de **0** paradas  
(campo `stop`).

```js
db.routes.countDocuments({ stops: { $gt: 0 } })
```
![stops](images/ud02-01-04f.png)
### g. Aeroportos de destino (1)
Obtén os aeroportos de destino dos voos que teñen como orixe o aeroporto **"KZN"**.

```js
db.routes.distinct("dst_airport", { src_airport: "KZN" })
```
![dst_airport](images/ud02-01-04g.png)

## 5. Colección `zips`

### a. Código postal sen habitantes (0,5)
Cal é o código postal da cidade de **ALLEN** que non ten habitantes.  
Só debe recuperarse o campo `zip` (sen `_id` nin outros campos).

```js
db.zips.find(
  { city: "ALLEN", pop: 0 },
  { _id: 0, zip: 1 }
)
```
![proxección 1](images/ud02-01-05a.png)

### b. Códigos postais máis poboados (0,5)
Listado cos **5 códigos postais máis poboados**  
(mostra os documentos completos).

```js
db.zips.find().sort({ pop: -1 }).limit(5)
```
![proxección 2](images/ud02-01-05b.png)

### c. Filtro con `$nor` (0,5)
Cantidade de documentos que **non** teñen menos de **5.000** habitantes  
nin máis de **1.000.000**.  
Debes utilizar o operador `$nor`.

```js
db.zips.countDocuments({
  $nor: [
    { pop: { $lt: 5000 } },
    { pop: { $gt: 1000000 } }
  ]
})
```
![proxección 3](images/ud02-01-05c.png)

### d. Comparación habitantes / zip (0,5)
Cantos documentos teñen máis habitantes que o seu propio código postal  
(campo `zip`).

```js
db.zips.countDocuments({
  $expr: { $gt: ["$pop", { $toInt: "$zip" }] }
})
```
![proxección 4](images/ud02-01-05d.png)

## 6. Colección `posts`

### a. Etiquetas restaurant ou moon (1)
Cantos mensaxes teñen as etiquetas **restaurant** ou **moon**.

```js
db.posts.countDocuments({
  tags: { $in: ["restaurant", "moon"] }
})
```
![arrays 1](images/ud02-01-06a.png)

### b. Mensaxes con comentarios de Salena Olmos (1)
As mensaxes que teñen algún comentario escrito polo usuario  
**Salena Olmos**.

```js
db.posts.find({
  "comments.name": "Salena Olmos"
})
```
